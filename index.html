Fix mysqldump: Unknown table 'COLUMN_STATISTICS' in information_schema

Disable column statistics by set the --column-statistics to zero. mysqldump --column-statistics=0 --host=<hostname> --user=<username> --password=<secret> db_name > backup_db_name.sql
